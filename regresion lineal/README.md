# Regresión lineal

En esta carpeta irán **ejemplos de regresión lineal ordinaria** (mínimos cuadrados): ajuste de una recta o hiperplano para predecir una variable continua a partir de variables explicativas.

## Qué encontrarás aquí

- Notebooks o scripts que carguen datos (desde `../DB` si aplica), entrenen el modelo y evalúen error (MAE, MSE, R²).

## Requisitos típicos

- Python con `numpy`, `pandas` y `scikit-learn` (u otra librería que indique el docente).

## Orden sugerido

1. Revisar el enunciado o el notebook principal.
2. Colocar los datos en `DB` si la práctica lo requiere.
3. Ejecutar celda a celda y contrastar resultados con lo esperado en clase.
