# Árboles de decisión

> Ortografía académica: **árboles de decisión**. El nombre de la carpeta se mantiene por compatibilidad con el repositorio.

Material sobre **árboles de decisión** para clasificación y/o regresión: particiones del espacio de características mediante reglas del tipo “si … entonces …”.

## Qué encontrarás aquí

- Entrenamiento de árboles, interpretación visual (si se incluyen gráficos), y métricas según el tipo de problema.

## Requisitos típicos

- `scikit-learn` (`DecisionTreeClassifier` / `DecisionTreeRegressor`), `pandas`, posiblemente `matplotlib` para visualizar el árbol.

## Datos

Usar la carpeta **`DB`** para conjuntos de datos comunes.
