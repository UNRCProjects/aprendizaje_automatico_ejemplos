# Aprendizaje automático — ejemplos

Este repositorio agrupa **ejemplos prácticos** de distintos algoritmos y técnicas de aprendizaje automático.

## Cómo usar este repositorio

1. **Clona o descarga** el repositorio en tu equipo.
2. **Abre la carpeta del tema** que toque en clase o en la práctica (por ejemplo `regresion lineal`, `k-Means`, etc.).
3. **Lee primero el `README` de esa carpeta**: indica el objetivo del ejemplo, requisitos y el orden sugerido de los archivos.
4. **Datos compartidos**: si un ejercicio necesita una base de datos, colócala en la carpeta **`DB`** siguiendo las instrucciones de `DB/README.md` y del archivo `DB/como_agregar_datos.md`
