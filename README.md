# Aprendizaje automático — ejemplos

Este repositorio agrupa **ejemplos prácticos** de distintos algoritmos y técnicas de aprendizaje automático. Cada tema vive en su **carpeta** con un `README` que resume qué verás ahí y cómo orientarte.

## Cómo usar este repositorio (alumnos)

1. **Clona o descarga** el repositorio en tu equipo.
2. **Abre la carpeta del tema** que toque en clase o en la práctica (por ejemplo `regresion lineal`, `k-Means`, etc.).
3. **Lee primero el `README` de esa carpeta**: indica el objetivo del ejemplo, requisitos y el orden sugerido de los archivos.
4. **Datos compartidos**: si un ejercicio necesita una base de datos, colócala en la carpeta **`DB`** siguiendo las instrucciones de `DB/README.md` y del archivo `DB/como_agregar_datos.md`.
5. **Entorno Python**: cuando aparezcan notebooks o scripts, usa un entorno virtual (`venv`) e instala las dependencias que indique cada práctica (o un `requirements.txt` si se añade en el futuro).
6. **Dudas**: revisa el `README` de la carpeta del tema y el general; si algo no está documentado, pregunta al docente para actualizarlo.

| Carpeta | Contenido típico |

| `K-Vecinos cercanos` | k-NN |
| `Navie Bayes` | Naive Bayes |

Los nombres de carpetas se mantienen como están en el repositorio para no romper enlaces; en los README de cada tema se usa la ortografía académica habitual (por ejemplo “árboles”, “Naive Bayes”).
