# Máquinas de vectores de soporte (SVM)

> En la carpeta: “Maquina con Soporte Vectorial” = **SVM / Support Vector Machine**.

Material sobre **SVM** para clasificación (y en algunos casos regresión con SVR): hiperplanos de separación óptima y truco del kernel.

## Qué encontrarás aquí

- `SVC`, `SVR` o equivalentes, elección de kernel (`linear`, `rbf`, etc.) y parámetros como `C` y `gamma`.

## Requisitos típicos

- `scikit-learn`, escalado de características (`StandardScaler`) recomendado en muchos casos.
