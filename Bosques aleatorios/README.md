# Bosques aleatorios (Random Forest)

Ejemplos del método **Random Forest**: conjunto de muchos árboles entrenados con **muestreo bootstrap** y **subconjuntos aleatorios de características**, que promedian o votan la predicción final.

## Qué encontrarás aquí

- `RandomForestClassifier` o `RandomForestRegressor`, importancia de variables y ajuste básico de hiperparámetros.

## Requisitos típicos

- `scikit-learn`, `pandas`, `numpy`; a veces `matplotlib` o `seaborn` para gráficos.

## Datos

Colocar archivos en **`DB`** según las instrucciones de cada práctica.
