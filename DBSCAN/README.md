# DBSCAN

Material sobre **DBSCAN** (agrupamiento basado en densidad): encuentra clusters de forma arbitraria y marca **ruido** como puntos no asignados.

## Qué encontrarás aquí

- `DBSCAN` de `scikit-learn`, parámetros `eps` y `min_samples`, visualización de clusters en 2D cuando sea posible.

## Requisitos típicos

- `scikit-learn`, `numpy`, `matplotlib` para gráficos de dispersión.

## Datos

Puntos o tablas en **`DB`** si el ejercicio usa un archivo externo.
