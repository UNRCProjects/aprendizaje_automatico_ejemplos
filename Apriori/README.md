# Apriori

Material sobre el algoritmo **Apriori** para **reglas de asociación** y itemsets frecuentes en datos transaccionales (cesta de la compra, etc.).

## Qué encontrarás aquí

- Minería de itemsets frecuentes y reglas del tipo “si A entonces B” con soporte y confianza.

## Requisitos típicos

- Puede usarse `mlxtend` (`apriori`, `association_rules`) u otra librería que indique el docente; a veces implementación didáctica manual.

## Datos

Transacciones en formato lista o CSV en **`DB`** según el formato que pida el notebook.
