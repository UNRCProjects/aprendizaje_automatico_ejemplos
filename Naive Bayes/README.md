# Naive Bayes

> Ortografía habitual: **Naive Bayes** (el nombre de la carpeta usa “Navie” por el repositorio).

Ejemplos del clasificador **ingenuo de Bayes**, que asume independencia condicional entre características dada la clase (hipótesis “ingenua”).

## Qué encontrarás aquí

- `GaussianNB`, `MultinomialNB` o `BernoulliNB` según el tipo de datos (continuos, conteos, binarios).

## Requisitos típicos

- `scikit-learn`, `pandas` para cargar tablas de texto o numéricas.

## Datos

Usar **`DB`** para datasets compartidos.
